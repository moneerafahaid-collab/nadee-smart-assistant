# مساعد نضيء الذكي

نظام داخلي لمنسوبي أمانة المنطقة ضمن إدارة البيانات. المساعد يجيب عن استفسارات **المؤشر الوطني للبيانات (نضيء) — دورة القياس الثالثة** وفق دليل سدايا، مع تحليل نضج المجالات الأربعة عشر.

التصميم مستوحى من هوية الدليل: كحلي داكن، تدرج أزرق–أخضر، وبطاقات العناصر الثلاثة (نضج / امتثال / تميز تشغيلي).

## التقنيات

- Python
- Django
- SQLite

## التشغيل محلياً

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_nadee
python manage.py runserver
```

ثم افتح: http://127.0.0.1:8000/

بيانات المعرفة تُحمَّل من `data/nadee_seed.json`.

## الاستضافة (من GitHub)

GitHub Pages لا يشغّل تطبيقات Django. انشر المشروع من المستودع عبر **Render**:

1. ادخل: https://dashboard.render.com
2. سجّل الدخول بـ **GitHub** واربط المستودع `nadee-smart-assistant`
3. اختر **New → Blueprint** أو **Web Service** من نفس المستودع
4. Render يقرأ `render.yaml` تلقائياً ويكمل النشر
5. بعد انتهاء البناء ستحصل على رابط مثل: `https://nadee-smart-assistant.onrender.com`

حساب الدخول بعد النشر: `employee` / `Nadee@123`

## الأقسام

1. **المساعد الذكي**: أسئلة عن المجالات، المعايير، مستويات النضج، والمرفقات.
2. **قاعدة المعرفة**: محتوى دليل نضيء (25+ مادة).
3. **تحليل الأداء**: جدول 14 مجالاً و42 معياراً، توقعات، عوامل، وتنبيهات.
